Распакуйте содержимое в любой каталог и запустите файл "1.Install.bat"
На все вопрос необходимо нажимать "Y", скрипт сам запустит установщики. Прокликиваем "Далее". По завершению установки потребуется перезагрузка.
После перезагрузки запускаем скрипт "2.settings.bat".
После этих шагов можно считать настройку пройденной.
